# AI & LLM Resources
My favorite resources for AI and LLMs. This is a work in progress.

## Websites/Blogs
 * [Anthropic Blog](https://www.claude.com/blog) - Anthropic's blog for (primarily) Claude Code.
 * [HuggingFace](https://huggingface.co/) - Huggingface's site for community driven model and dataset development.

## Courses/Tutorials
 * [Anthropic Academy](https://www.anthropic.com/learn) - Courses and tutorials for working with Anthropic and Claude.
 * [HuggingFace Learn](https://huggingface.co/learn) - HuggingFace's page for a variety of AI, LLM, and ML courses.
 * [OpenAI Academy](https://academy.openai.com/) - OpenAI's hub for a variety of content

## Webinars/Events
 * [Anthropic Events](https://www.anthropic.com/events) - Anthropic's page for webinars and in-person events
 * [OpenAI Events](https://academy.openai.com/public/events) - OpenAI's page for webinars and in-person events
